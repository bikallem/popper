Module Popper.Consumed

  • Types
  • Functions

This module defines a type that represents the output of running a sample function ( an 'a Sample.t value) on some input. The ouput is a a record of the input stream consumed with tags indicating the various sub-sections of the stream that were used for sampling different types of values
